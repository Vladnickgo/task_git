Удалённые ссылки — это ссылки (указатели) в ваших удалённых репозиториях, включая
ветки, теги и так далее. 
Полный список удалённых ссылок можно получить с помощью
команды git ls-remote <remote> или команды git remote show <remote> для получения
удалённых веток и дополнительной информации. 
Тем не менее, более распространенным способом является использование веток слежения.
88Ветки слежения — это ссылки на определённое состояние удалённых веток. 
Это локальные ветки, которые нельзя перемещать; Git перемещает их автоматически при любой
коммуникации с удаленным репозиторием, чтобы гарантировать точное соответствие с
ним. 
Представляйте их как закладки для напоминания о том, где ветки в удалённых
репозиториях находились во время последнего подключения к ним.
